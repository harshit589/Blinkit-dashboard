 # Blinkit-dashboard
 🛒 Blinkit Sales Analysis Dashboard | Power BI Project
📌 Project Overview

This project is an interactive Blinkit Sales Analysis Dashboard developed in Power BI to analyze sales performance, outlet performance, item distribution, customer ratings, and inventory insights.
The dashboard helps in understanding key business metrics and identifying trends across different outlet types, outlet sizes, and product categories.

📊 Dashboard Preview


🎯 Project Objectives
Analyze overall sales performance of Blinkit outlets.
Track average sales, ratings, and number of items.
Compare sales based on outlet size and outlet type.
Identify top-performing item categories.
Analyze fat content distribution and outlet establishment trends.
Create an interactive and visually appealing dashboard for business insights.

📊 Key Insights
Supermarket Type 1 generated the highest sales among all outlet types.
Fruits & Vegetables and Snack Foods contributed the highest sales.
Medium-sized outlets showed strong overall performance.
Tier 3 outlets generated higher sales compared to Tier 1 and Tier 2.
Low Fat products contributed more to total sales.
